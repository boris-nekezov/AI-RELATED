# Cursor Tab Control - Disable/Enable Cursor Tab

## Disable Cursor Tab

Отворете Command Palette (`Ctrl + Shift + P` / `Cmd + Shift + P`) и изпълнете командата `Disable Cursor Tab` за деактивиране на табове в редактора.

## Enable Cursor Tab

Отворете Command Palette (`Ctrl + Shift + P` / `Cmd + Shift + P`) и изпълнете командата `Enable Cursor Tab` за активиране на табове в редактора.
